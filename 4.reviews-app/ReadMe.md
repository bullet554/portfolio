# Приложение для управления отзывами о товарах

## Описание проекта
Это полностековое веб-приложение, разработанное для демонстрации навыков работы с **Node.js**, **Express** и **JavaScript**. Проект позволяет пользователям просматривать и оставлять отзывы о различных товарах.

## Основные технологии
* **Бэкенд**: Node.js, Express
* **Фронтенд**: HTML, CSS (SCSS), JavaScript
* **Хранение данных**: JSON-файл

## Функционал
* Просмотр списка товаров с отзывами
* Добавление новых отзывов
* Валидация длины текста отзыва (50-500 символов)
* Сохранение данных в JSON-файл
* Адаптивный дизайн

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone 
```
2. Установите зависимости:
```
npm install
```
3. Запустите сервер:
```
node start
```
4. Откройте в браузере:
```
http://localhost:3000
```
