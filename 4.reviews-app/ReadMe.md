# Приложение для управления отзывами о товарах

## Описание проекта
**Полностековое веб-приложение** для работы с отзывами о товарах. Проект демонстрирует навыки разработки на **Node.js** и **Express**, а также взаимодействие между фронтендом и бэкендом. Приложение позволяет пользователям просматривать товары и оставлять о них отзывы.

## Ключевые особенности
* **Двусторонняя связь** между клиентом и сервером
* **Валидация данных** на стороне клиента
* **Сохранение данных** в JSON-формате
* **Адаптивный интерфейс**
* **Система отзывов** с ограничениями по длине

## Технологии

* **Бэкенд**
    * **Node.js** — серверная платформа
    * **Express** — веб-фреймворк
    * **JavaScript** — язык программирования

* **Фронтенд**
    * **HTML5** — структура страницы
    * **CSS3** (SCSS) — стилизация
    * **JavaScript** — клиентская логика

## Функционал
* **Управление отзывами:**
    * Просмотр существующих отзывов
    * Добавление новых отзывов
    * Валидация длины текста (50-500 символов)
* **Хранение данных:**
    * Сохранение в JSON-файл
    * Загрузка данных при старте
* **Пользовательский интерфейс:**
    * Адаптивный дизайн
    * Интуитивно понятная навигация

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/bullet554/portfolio.git
```
2. Перейдите в директорию проекта:
```
cd ./4.reviews-app/
```
3. Установите зависимости:
```
npm install
```
4. Запустите сервер:
```
npm start
```
5. Откройте в браузере:
```
http://localhost:3000
```
