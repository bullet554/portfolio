# Веб-слайдер

## Описание проекта
**Интерактивный веб-слайдер** — это простое, но функциональное решение для отображения серии изображений на веб-странице. Проект демонстрирует базовые навыки работы с современными веб-технологиями и может быть легко адаптирован под различные задачи.

## Ключевые особенности
* **Интуитивно понятный интерфейс** с кнопками навигации и точками-маркерами
* **Плавные анимации** при переключении слайдов
* **Адаптивный дизайн** для корректного отображения на любых устройствах
* **Универсальная структура** кода, позволяющая легко добавлять новые изображения

## Основные технологии
* **HTML5** для создания структуры
* **SCSS** для стилизации и организации CSS-кода
* **JavaScript** для реализации логики работы слайдера

## Функционал
* **Навигация** между слайдами с помощью кнопок и точек-маркеров
* **Автоматическое определение** количества слайдов
* **Плавные переходы** между изображениями
* **Циклическое переключение** слайдов
* **Поддержка** различных форматов изображений

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/bullet554/portfolio.git
```
2. Перейдите в директорию проекта:
```
cd ./1.web-slider/
```
3. Откройте файл index.html в браузере
