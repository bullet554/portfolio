# Банковское приложение для управления счетами

## Описание проекта
**Полностековое веб-приложение**, имитирующее работу банковского сервиса. Проект демонстрирует навыки разработки на **Node.js** и **Express**, а также взаимодействие между фронтендом и бэкендом. Приложение позволяет пользователям регистрироваться и входить в аккаунт, и выполнять финансовые операции.

## Ключевые особенности
* **Система авторизации** с регистрацией и входом в систему
* **Управление счетами** с возможностью пополнения и снятия средств
* **Валидация данных** на стороне клиента и сервера
* **Адаптивный интерфейс**
* **Сохранение данных** в JSON-формате
* **Обработка ошибок** и валидация форм

## Технологии

* **Бэкенд**
    * **Node.js** — серверная платформа
    * **Express** — веб-фреймворк
    * **JavaScript** — язык программирования
    * **body-parser** — парсинг данных

* **Фронтенд**
    * **HTML5** — структура страницы
    * **CSS3** (SCSS) — стилизация
    * **JavaScript** — клиентская логика
    * **Fetch API** — работа с запросами

## Функционал
* **Управление пользователями:**
    * Регистрация новых пользователей
    * Авторизация и выход из системы при обновлении страницы
    * Личный кабинет
* **Управление счетами:**
    * Просмотр баланса
    * Пополнение счета
    * Снятие средств
* **Валидация данных:**
    * Проверка корректности ввода сумм
    * Проверка логинов и паролей
* **Хранение данных:**
    * Сохранение в JSON-файл
    * Загрузка данных при старте

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/bullet554/portfolio.git
```
2. Перейдите в директорию проекта:
```
cd ./5.bank-app/
```
3. Установите зависимости:
```
npm install
```
4. Запустите сервер:
```
npm start
```
5. Откройте в браузере:
```
http://localhost:3001
```
