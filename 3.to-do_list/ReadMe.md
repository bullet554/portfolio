# Список дел

## Описание проекта
**Интерактивный To-Do List** — это веб-приложение для управления задачами с возможностью добавления, удаления и сохранения данных в локальное хранилище браузера. Проект демонстрирует базовые навыки работы с DOM и локальным хранилищем.

## Ключевые особенности
* **Интуитивный интерфейс** для управления задачами
* **Автоматическое сохранение** данных
* **Динамическое обновление** списка задач
* **Простая структура** кода

## Технологии
* **HTML5** для создания структуры приложения
* **CSS3** (включая SCSS) для стилизации элементов
* **JavaScript** для реализации логики

## Функционал
* **Система добавления** задач через форму
* **Механизм удаления** задач
* **Хранение данных** в localStorage
* **Динамическое обновление** интерфейса
* **Валидация пользовательского** ввода

## Особенности реализации
* **Работа с DOM** для создания элементов
* **Обработка событий** клика
* **Работа с localStorage** для сохранения данных
* **Проверка ввода** на пустоту

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/bullet554/portfolio.git
```
2. Перейдите в директорию проекта:
```
cd ./3.to-do_list/
```
3. Откройте файл index.html в браузере

## Инструкция по использованию
1. Введите задачу в поле ввода
2. Нажмите кнопку “Добавить”
3. Управляйте списком задач
4. Данные сохраняются автоматически
