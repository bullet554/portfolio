# Приложение для работы с API шуток Чака Норриса

## Описание проекта
**Интерактивное веб-приложение** для получения случайных шуток через внешний API. Проект демонстрирует работу с Fetch API, обработку асинхронных запросов и управление состоянием в React.

## Ключевые особенности
* **Взаимодействие с API:**
    * Получение данных через Fetch API
    * Обработка ответов сервера
    * Управление состоянием загрузки
* **Пользовательский интерфейс:**
    * Адаптивный дизайн
    * Индикатор загрузки
    * Кнопка получения новой шутки
    * Отображение результата
* **Обработка ошибок:**
    * Обработка сетевых ошибок
    * Отображение сообщений об ошибках

## Технологии и инструменты

* **Frontend:**
    * **React.js** — для создания пользовательского интерфейса
    * **JavaScript** — основной язык разработки
    * **CSS** — стилизацию компонентов
* **API:**
    * **Fetch API** — для работы с внешними данными
    * **Chuck Norris API** — источник шуток

## Функционал
* **Получение шуток:**
    * Загрузка случайной шутки по клику
    * Автоматическое обновление состояния загрузки
    * Отображение результата
* **Интерфейс:**
    * Визуальный индикатор загрузки
    * Информирующие сообщения
    * Удобная кнопка управления

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/bullet554/portfolio.git
```
2. Перейдите в директорию проекта:
```
cd ./7.chuck_norris-api/
```
3. Установите зависимости:
```
npm install
```
4. Запустите приложение:
```
npm start
```
5. Откройте в браузере:
```
http://localhost:3000
```
