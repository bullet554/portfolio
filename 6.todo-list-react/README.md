# Приложение для управления задачами с REST API и Redux Toolkit

## Описание проекта
**Полнофункциональное веб-приложение** для управления списком задач с серверной частью на Node.js и клиентским интерфейсом на React. Проект демонстрирует работу с состоянием через Redux Toolkit, интеграцию с React Bootstrap и взаимодействие с REST API.

## Ключевые особенности
* **Клиент-серверное взаимодействие:**
    * CRUD операции через REST API
    * Обработка асинхронных запросов
    * Управление состоянием через RTK Query
    * Обработка ошибок на клиенте и сервере
* **Пользовательский интерфейс:**
    * Адаптивный дизайн с React Bootstrap
    * Система переключения тем оформления
    * Интуитивно понятная навигация
    * Валидация форм
* **Функциональные возможности:**
    * Создание и редактирование задач
    * Управление статусами выполнения
    * Фильтрация по категориям
    * Сохранение данных на сервере

## Технологии и инструменты

* **Frontend:**
    * **React.js** — современный JavaScript-фреймворк
    * **Redux Toolkit** — управление состоянием приложения
    * **React Bootstrap** — компоненты UI
    * **RTK Query** — работа с API
    * **SCSS** — стилизацию компонентов
* **Backend:**
    * **Node.js** — серверная платформа
    * **Express.js** — веб-фреймворк
    * **Axios** — HTTP-запросы
    * **CORS** — междоменное взаимодействие
* **Дополнительно:**
    * **TypeScript** — статическая типизация
    * **Git** — версионный контроль

## Функционал
* **Создание задач:**
    * Добавление новых элементов через форму
    * Автоматическая генерация ID
* **Управление списком:**
    * Отображение всех задач
    * Удаление отдельных элементов
    * Переключение статуса выполнения
* **Темы оформления:**
    * Светлая тема по умолчанию
    * Тёмная тема по запросу
    * Плавные переходы между темами
* **Фильтрация:**
    * Все задачи
    * Выполненные задачи
    * Активные задачи

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/bullet554/portfolio.git
```
2. Перейдите в директорию сервера:
```
cd ./6.todo-list-react/server/
```
3. Установите зависимости:
```
npm install
```
4. Запустите сервер:
```
npm run start
```
5. Перейдите в директорию клиента:
```
cd ..
cd ./client/
```
6. Установите зависимости:
```
npm install
```
7. Запустите сервер:
```
npm start
```
8. Откройте в браузере:
```
http://localhost:3000
```

## Архитектура проекта
* **Клиентская часть:**
    * Модульная структура компонентов
    * Отдельные слайсы для состояния
    * Хуки для работы с API
```
6.todo-list-react/client/
├── src/
│   ├── component/                # Компоненты интерфейса
│   │   ├── AddTodo.js
│   │   ├── ThemeSwitcher.js
│   │   ├── TodoFilter.js
│   │   └── TodoList.js
│   ├── contexts/                 # Контексты React
│   │   ├── ThemeContext.js
│   │   └── FilterContext.js
│   ├── slices/                   # Редуксы и слайсы
│   │   ├── themeSlice.js
│   │   └── todoSlice.js
│   ├── hoc/                     # Higher-Order Components
│   │   └── withFilter.js
│   ├── styles/                  # Стили
│   │   ├── index.scss
│   │   └── ...
│   ├── App.js                   # Главный компонент
│   ├── store.js                 # Конфигурация Redux
│   └── index.js                 # Точка входа
├── public/                      # Публичные файлы
│   └── index.html
├── package-lock.json
└── package.json                 # Зависимости клиента
```
* **Серверная часть:**
    * REST API endpoints
    * Middleware для обработки запросов
    * Система ошибок
    * CORS конфигурация
```
server/
├── index.js                   # Точка входа сервера
├── tasks.json                # Файл хранения задач
├── package.json              # Зависимости сервера
└── package-lock.json
```
