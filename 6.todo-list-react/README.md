# Приложение для управления задачами с темой оформления

## Описание проекта
**Интерактивное веб-приложение** для управления списком задач с возможностью переключения между светлой и тёмной темой оформления. Проект демонстрирует работу с состоянием в React, Redux и интеграцию с React Bootstrap.

## Ключевые особенности
* **Управление задачами:**
    * Добавление новых задач
    * Удаление существующих задач
    * Переключение между темами оформления
    * Фильтрация задач по статусу
* **Пользовательский интерфейс:**
    * Форма ввода задач
    * Визуальный список элементов
    * Кнопки управления
    * Переключатель темы оформления
* **Система состояний:**
    * Redux для управления глобальным состоянием
    * Контекстный API для фильтрации

## Технологии

* **React.js** — для создания пользовательского интерфейса
* **React Bootstrap** — для стилизации компонентов
* **Redux** — для управления состоянием
* **SCSS** — для стилизации
* **JavaScript** — язык программирования

## Функционал
* **Создание задач:**
    * Добавление новых элементов через форму
    * Автоматическая генерация ID
* **Управление списком:**
    * Отображение всех задач
    * Удаление отдельных элементов
    * Переключение статуса выполнения
* **Темы оформления:**
    * Светлая тема по умолчанию
    * Тёмная тема по запросу
    * Плавные переходы между темами
* **Фильтрация:**
    * Все задачи
    * Выполненные задачи
    * Активные задачи

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/bullet554/portfolio.git
```
2. Перейдите в директорию проекта:
```
cd ./6.todo-list-react/
```
3. Установите зависимости:
```
npm install
```
4. Запустите сервер:
```
npm start
```
5. Откройте в браузере:
```
http://localhost:3000
```
