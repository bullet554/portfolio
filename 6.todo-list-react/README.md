# Приложение для управления задачами (Todo List)

## Описание проекта
**Интерактивное веб-приложение** для управления списком задач с сохранением данных в localStorage. Проект демонстрирует работу с состоянием в React и интеграцию с React Bootstrap.

## Ключевые особенности
* **Управление задачами:**
    * Добавление новых задач
    * Удаление существующих задач
    * Сохранение списка в localStorage
    * Автоматическое восстановление данных
* **Пользовательский интерфейс:**
    * Форма ввода задач
    * Визуальный список элементов
    * Кнопки управления
* **Валидация:**
    * Проверка на пустые строки
    * Корректная обработка данных

## Технологии

* **React.js** — для создания пользовательского интерфейса
* **React Bootstrap** — для стилизации компонентов
* **localStorage** — для хранения данных
* **JavaScript** — язык программирования

## Функционал
* **Создание задач:**
    * Добавление новых элементов через форму
    * Автоматическая генерация ID
* **Управление списком:**
    * Отображение всех задач
    * Удаление отдельных элементов
* **Сохранение данных:**
    * Автоматическое сохранение при изменении списка
    * Загрузка данных при перезагрузке
* **Интерфейс:**
    * Адаптивный дизайн
    * Интуитивно понятная навигация

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/bullet554/portfolio.git
```
2. Перейдите в директорию проекта:
```
cd ./6.todo-list-react/
```
3. Установите зависимости:
```
npm install
```
4. Запустите сервер:
```
npm start
```
5. Откройте в браузере:
```
http://localhost:3000
```
