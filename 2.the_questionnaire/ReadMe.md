# Опросник

## Описание проекта
**Интерактивный опросник** — это простое веб-приложение для сбора ответов пользователей через форму с радио-кнопками. Проект демонстрирует базовые навыки работы с формами и обработкой пользовательских данных.

## Ключевые особенности
* **Интуитивный интерфейс** с понятной навигацией
* **Валидация** ответов перед отправкой
* **Мгновенная обратная связь** с отображением результатов
* **Простая структура** кода для легкого понимания

## Основные технологии
* **HTML5** для создания структуры формы
* **CSS3** для стилизации элементов
* **JavaScript** для обработки данных и логики

## Функционал
* **Интерактивные формы** с радио-кнопками
* **Система валидации** для проверки заполнения
* **Обработка данных** и отображение результатов
* **Пользовательский интерфейс** с визуальной обратной связью

## Установка и запуск
1. Клонируйте репозиторий:
```
git clone https://github.com/bullet554/portfolio.git
```
2. Перейдите в директорию проекта:
```
cd ./2.the_questionnaire/
```
3. Откройте файл index.html в браузере

## Инструкция по использованию
1. Откройте index.html в браузере
2. Выберите ответы на вопросы из предложенных вариантов
3. Нажмите кнопку "Завершить опрос"
4. Ознакомьтесь с результатами
