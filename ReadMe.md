# О портфолио

Добро пожаловать в мое портфолио! Здесь представлены проекты, над которыми я работал, демонстрирующие мои навыки и опыт в веб-разработке.

## Список проектов

1. **Веб-слайдер**  
    * **Описание:** Интерактивный слайдер для отображения изображений с возможностью переключения между ними.  
    * **Технологии:** HTML, CSS, JavaScript
    * **Путь:** ./1.web-slider/

2. **Опросник**  
    * **Описание:** Веб-приложение для сбора и обработки ответов пользователей через форму.  
    * **Технологии:** HTML, CSS, JavaScript
    * **Путь:** ./2.the_questionnaire/

3. **Список дел**  
    * **Описание:** Интерактивный менеджер задач с возможностью добавления, удаления и сохранения данных.  
    * **Технологии:** HTML, CSS, JavaScript, localStorage
    * **Путь:** ./3.to-do_list/

4. **Приложение для управления отзывами**  
    * **Описание:** Полностековое веб-приложение для работы с отзывами о товарах.  
    * **Технологии:** Node.js, Express, HTML, CSS, JavaScript
    * **Путь:** ./4.reviews-app/

5. **Банковское приложение для управления счетами**  
    * **Описание:** Полностековое банковское веб-приложение с системой авторизации и управления финансами.  
    * **Технологии:** Node.js, Express, body-parser, HTML, CSS, JavaScript, Fetch API
    * **Путь:** ./5.bank-app/

6. **Todo List Fullstack Application**  
   * **Описание:** Полнофункциональное веб-приложение с серверной частью на Node.js и интерфейсом на React.  
   * **Технологии:** React.js, Redux Toolkit, React Bootstrap, Node.js, Express.js, Axios, SCSS, TypeScript, RTK Query
   * **Путь:** ./6.todo-list-react/

7. **Chuck Norris Jokes API**  
   * **Описание:** Приложение для получения случайных шуток с использованием внешнего API.  
   * **Технологии:** React, JavaScript, CSS, Fetch API
   * **Путь:** ./7.chuck_norris-api/

8. **Online Store (E-commerce приложение)**  
   * **Описание:** Полнофункциональное интернет-приложение магазина с каталогом товаров, фильтрацией, пагинацией, страницей товара и корзиной. Реализована авторизация пользователей, добавление товаров в корзину и взаимодействие с серверным API.
   * **Технологии:** React.js, React Router, Context API, SCSS, Node.js, Express.js, REST API, Supabase
   * **Путь:** ./8.online-store-app/
